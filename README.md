# 💼 JobHunt - Full Stack Job Portal Application

A modern full-stack job portal application built using the MERN stack. JobHunt enables recruiters to post jobs, manage applicants, and allows candidates to search and apply for jobs through a secure and responsive platform.

---

## 🚀 Features

### 👨‍💻 Candidate Features
- User Registration & Login
- Browse available jobs
- Search jobs by keywords
- Apply for jobs
- View applied jobs
- Update profile information
- Resume upload support

### 🏢 Recruiter Features
- Recruiter authentication
- Company management
- Create job postings
- Manage job listings
- View applicants
- Track applications

### 🔐 Authentication & Security
- JWT Authentication
- Protected Routes
- Role-based Authorization
- Secure API handling

### 🎨 Modern UI
- Responsive design
- ShadCN UI components
- Clean dashboard interface
- Mobile-friendly layout

### ⚡ State Management
- Redux Toolkit for global state management

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/TusharChaudhari5435/Job-Portal-App.git
```

### Backend Setup

```bash
cd Backend
npm install
npm run dev
```

### Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the **Backend** folder.

```env
PORT=
MONGO_URI=
JWT_SECRET=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
```
